### 專案說明
當今天帶毛小孩去看病，但他卻是第一次來這間診所看病，診所並沒有他的資料，那怎麼辦？？  
1. 提供毛小孩統一的醫療紀錄的儲存平台
2. 為浪浪們提供一個與飼主間的認養媒合平台
3. 提供飼主們可以查詢寵物飼養資訊的平台
